# 🍕 Karachi Pizza & Fast Food - POS System

A premium, full-featured Restaurant Point of Sale (POS) & Management System built with React.js. Designed for internal restaurant management with a modern black, orange, and white theme.

## ✨ Features

### 🔐 Secure Login
- Manager-only access
- Protected dashboard

### 📊 Dashboard
- Real-time sales analytics
- Order statistics (Pending, Completed, Cancelled)
- Payment method breakdown
- Daily/Weekly/Monthly sales charts
- Best-selling items tracking

### 💳 POS Billing
- Touch-friendly interface
- Search and browse menu by categories
- Add items with quantity control
- Extra options (Cheese, Sauce, Toppings)
- Cooking notes for kitchen
- Automatic calculation:
  - Subtotal
  - Tax (10%)
  - Discount
  - Service Charge
  - Grand Total
- Multiple order types:
  - Dine In
  - Take Away
  - Delivery
- Multiple payment methods:
  - Cash
  - Debit Card
  - Credit Card
  - Bank Transfer
  - Mobile Wallet

### 📋 Menu Management
Pre-loaded with complete menu:
- 🍕 Pizza (8 varieties + toppings)
- 🍔 Burgers (9 varieties)
- 🍖 BBQ (10 items)
- 🥘 Karahi (6 items)
- 🍲 Handi (6 items)
- 🥤 Shakes (4 flavors)
- 🍦 Ice Cream (3 varieties)
- 🍧 Falooda
- 🍛 Biryani & Pulao (6 items)
- 🌯 Rolls & Shawarma (5 items)
- 🍟 French Fries (5 varieties)

### 👥 Customer Management
- Store customer details
- Order history tracking
